SPACE LINK GENERATORS — HELP IMAGE SLOTS
========================================
Drop your NMS screenshots into this "help" folder. Both tools
(System Atlas + Base Creator) read from this one folder.

NAMING — this is the important bit:
  Every screenshot is named   <field>-<number>.jpg
  starting at 1, counting up. Examples:
     region-1.jpg         (one picture for Region)
     star_class-1.jpg     (one picture for Star class)
     glyphs-1.jpg  glyphs-2.jpg  glyphs-3.jpg   (THREE pictures, shown in order)

MULTIPLE PICTURES: just keep numbering up. region-1, region-2, region-3...
  They appear top-to-bottom in the panel, in number order. Up to 6 each.
  Number them with NO gaps (1,2,3 — don't skip to 1,3).
  A slot with no file just doesn't show — nothing breaks.

Any image you haven't added yet stays hidden; the text steps still show.
Save as JPG to keep files small (the tool also accepts PNG, but PNGs are ~6x
bigger and GitHub's web uploader chokes on the batch). Crop tight + circle/
arrow the value (see KolourPaint).

(# = a number: 1 for a single picture, or 1, 2, 3... for several.)

HOW TO UPLOAD: on GitHub, "Add file -> Upload files", then drag this whole
"help" folder in (folder structure is preserved), and commit.
